# Play
Скачайте репозиторий и разархивируйте
Открыть эту папку в VS Code 
Далее в терминале
py -m venv venv

В терминале установить библиотеку pygame
.\venv\Scripts\pip.exe install pygame

Далее запускаем
./venv/Scripts/python.exe puzzle.py
